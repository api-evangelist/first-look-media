# First Look MediaThis is a repo for managing machine readable artifacts for the [First Look Media APIs](https://firstlook.org/theintercept/). The goal is to maintain and evolve an OpenAPI, Postman Collection, and APIs.json index for the [First Look Media APIs](https://firstlook.org/theintercept/).They are a constant work in progress, and will be updated on a regular basis when new information is added, or more of the specifications are supported.Feel free to submit a pull request to evolve any of the artifacts, and leave GitHub issues if you have any questions, comments, or feedback.- [Website](https://firstlook.org/theintercept/)- [Developer](https://firstlook.org/theintercept/)- [Twitter](https://twitter.com/firstlookmedia)The [APIs.json](https://github.com/api-evangelist/first-look-media/blob/master/apis.json) index provides links to all the relevant links for the First Look Media APIs, including the OpenAPI and Postman if they are present.
